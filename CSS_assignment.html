  1. What are the benefits of using CSS?
    2. ➤Some of the benefits of using CSS are:
    • It makes the website more attractive and user-friendly by applying different styles, colors, fonts, and formats to the HTML elements.
    • It reduces the amount of code and improves the performance and loading speed of the website by applying one style rule to multiple elements or pages.
    • It makes the website more consistent and easier to maintain by separating the content from the presentation and allowing changes to be made in one place.
    • It enhances the accessibility and compatibility of the website by following web standards and allowing different devices and browsers to display the website correctly.
  2. What are the disadvantages of CSS?
    3. ➤
    1. CSS can create confusion among web browsers. What works with one browser might not always work with another. The web developers need to test for compatibility, running the program across multiple browsers.
    2. There exists a scarcity of security.
    3. Different levels of CSS i.e. CSS, CSS 2, CSS 3 are often quite confusing.
    4. Browser compatibility (some styles sheet are supported and some are not).
  3. What is the difference between CSS2 and CSS3?
    4. ➤
    • CSS3 is divided into many different modules, each with its own features and specifications.
    • Browser support: CSS3 supports more browsers than CSS2.
    • Selectors: CSS3 introduces several new selectors.
    • onts: CSS3 allows designers to use special fonts, like those available in Google Fonts and Typecast.
  4. Name a few CSS style components?
    5. ➤
    • Selector: class name, id name or element name that is target
    • Attribute: name of the attribute you want to style for example border, color,background, position etc.
    • Value of Property: value that will be assigned to attribute.
  5. What do you understand by CSS opacity?
    6. ➤In CSS, opacity is a property that describes the transparency of an element. It's the opposite of transparency and represents the degree to which content is hidden behind an element.
  6. How can the background color of an element be changed?
    7. ➤To change the background color of an element, you can use the CSS background-color property. You can set the color name or code you want and place it inside a style attribute. Then add this style attribute to an HTML element, like a table, heading, div, or span tag.
  7. How can image repetition of the backup be controlled?
    8. ➤This task can be achieved by using the background-repeat property that will help us to control the repetition of the image. The background-repeat property in CSS is used to repeat the background image both horizontally and vertically. It also decides whether the background image will be repeated or not.
  8. What is the use of the background-position property?
    9. ➤The background-position property in CSS sets the starting position of a background image. It allows you to move a background image (or gradient) around within its container. for example:-
    1. Length values: (e.g. 100px 5px)
    2. Percentages: (e.g. 100% 5%)
  9. Which property controls the image scroll in the background?
    10. ➤The background-attachment property in CSS controls the scrolling of an image in the background. This property can be applied to all HTML elements.
  10. Why should background and color be used as separate properties?
    11. ➤The background property is a complex property in CSS, and if it is combined with color, the complexity will further increase.
  11. How to center block elements using CSS1?
    12. ➤To centrally align the block elements, we can simply make use of the (center) tag. All the elements within the (center) tag will be centrally aligned.

  18. What are the media types allowed by CSS?
    19. ➤Continuous or paged Visual, audio, speech, or tactile Grid (for character grid devices), or bitmap Interactive (for devices that allow user interaction), or static (for those that do not) All (includes all media types) Some specific media types include:
    1. Print: Used for printers
    2. Projection: Used for projected presentations, for example projectors
    3. Screen: Used primarily for color computer screens
    4. Tty: Used for media using a fixed-pitch character grid, such as teletypes, terminals, or portable devices with limited display capabilities
    5. Speech: Used for screenreaders that "reads" the page out loud
    6. Aural: Intended for speech synthesizers
    7. Braille: Intended for braille tactile feedback devices
    8. Embossed: Intended for paged braille printers.
